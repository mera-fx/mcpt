DASHBOARD MONOCHROME THEME AND NEW-TAB REPORT LINKS
===================================================

Changes
-------
1. The Open primary report button now uses:
   target="_blank" rel="noopener noreferrer"

2. The dashboard changes from the blue theme to a reference-inspired palette:
   - near-black page background
   - charcoal panels
   - warm off-white text
   - muted stone secondary text
   - subtle bronze accent
   - restrained status colours
   - low-contrast grid background

Scope
-----
This changes the quantitative research dashboard generated by
build_research_dashboard_v2.py. It does not restyle the separate primary
research reports.

Safety
------
The patch and dashboard rebuild read saved research files only.
They do not run strategies, optimization, MCPT, bootstrap, paper simulation,
live trading, or market-data requests.
