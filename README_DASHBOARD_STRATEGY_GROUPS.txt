RESEARCH DASHBOARD STRATEGY GROUPS
==================================

Purpose
-------
Organize EXP-001 through EXP-014 into five collapsed research lineages while
keeping them under the Strategy research section.

Groups
------
1. BTCUSDT Hourly Strategy Ideas
   EXP-001, EXP-002, EXP-003
   These share market/timeframe but are separate strategy hypotheses.

2. ORB Locked-Transfer Lineage
   EXP-004 -> EXP-005 -> EXP-006

3. Long-Only ORB Exit-Geometry Lineage
   EXP-007 -> EXP-008

4. Multi-Strategy Discovery and Opening-Drive Lineage
   EXP-009 -> EXP-010 -> EXP-011

5. Extended-Hours Context Lineage
   EXP-012 -> EXP-013 -> EXP-014

Dashboard behaviour
-------------------
- Each research group starts collapsed.
- Each experiment inside a group also starts collapsed.
- Expand all and Collapse all control both levels.
- Search opens matching groups and experiments.
- Overview links open the enclosing group and selected experiment.
- The overview table includes a Research group column.

Safety
------
The patch changes dashboard presentation and tests only. It does not rerun
strategies, optimization, MCPT, bootstrap, paper testing or market-data work.
